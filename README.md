# Bio-Headband 1.0: AI-Powered Neurotech System

**An AI-powered Bio-Headband system for sleep phase monitoring (EEG/HRV). Features C++ firmware with digital signal filtering (Butterworth), RMSSD analysis, and a Python Telegram bot integrated with OpenAI GPT-4 for automated hypnogram generation and health insights. Zero-EMF compliant design for biological safety.**

##  Key Features
* **Real-time Data Acquisition:** Captures EEG and accelerometer data.
* **Advanced Signal Processing:** Utilizes Butterworth filtering and RMSSD analysis.
* **Intelligent Assistant:** Python-based Telegram bot powered by OpenAI GPT-4.
* **Data Documentation:** See PDF reports for full methodology.
